# Discord Self Message Cleaner

作者：mzrodyu / catie

一个 Chrome 扩展，以**悬浮面板**的形式注入到 Discord 网页端，批量删除当前账号自己的消息。

## 功能

- **悬浮面板**：直接悬浮在 Discord 页面上，可拖拽移动，无需弹窗
- **Token 自动获取**：打开面板时自动从 Discord 内部提取 token，无需手动填写
- **一键填充 ID**：点「当前」按钮自动从当前页面 URL 读取服务器和频道 ID
- **预览后删除**：先预览自己的消息，确认后再批量删除，避免误删
- **时间范围过滤**：支持按起始/结束时间筛选
- **限速处理**：触发 Discord 429 限速时自动等待重试
- **免 Token 模式**：通过模拟 UI 点击删除当前可见消息，不需要 token

## 安装

适用于 Chrome、Edge、Brave 等 Chromium 系浏览器。

1. 下载或 clone 本仓库到本地
2. 打开浏览器扩展管理页（Chrome：`chrome://extensions/`）
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹（含 `manifest.json` 的目录）
5. 打开 `discord.com`，点击右上角扩展图标即可显示/隐藏悬浮面板

修改项目文件后，需要在扩展管理页点「重新加载」。

## 使用

1. 打开任意 Discord 频道页面
2. 点击扩展图标，悬浮面板出现，token 自动填入
3. 点「当前」按钮填充服务器 ID 和频道 ID（或手动填写）
4. 设置最多删除数量、间隔毫秒、可选时间范围
5. 勾选免责声明
6. 点「预览」确认消息列表，再点「删除预览」执行

> 免 Token 模式：直接点「免Token删除」，扩展会模拟点击当前页面可见的自己的消息依次删除，无需 token 和 ID。

## 注意

- 使用自动化工具操作账号违反 Discord 服务条款，风险自负
- 删除不可恢复，建议先用小数量（如 10 条）测试
- Token 不会持久化保存，但仍建议只在自己的可信设备上使用
- 普通用户只能删除自己的消息

