# Discord 自助冲水机 (Discord Self Message Cleaner)

**作者**: mzrodyu / catie

这是一个用于自动、批量删除 Discord 中自己历史消息的 Chrome 扩展。

---

## 安装

支持 Chrome, Edge, Brave 等所有 Chromium 内核浏览器。

1. 把这个项目下载或 clone 到你的电脑上。
2. 打开浏览器的扩展管理页面 (Chrome 浏览器输入 chrome://extensions/)。
3. 打开右上角的「开发者模式」。
4. 点击「加载已解压的扩展程序」，选择你下载的文件夹（包含 manifest.json 文件）。
5. 在 Discord 网页版点击右上角插件栏的图标即可弹出悬浮面板。

---

## 使用

1. 在浏览器打开任意 Discord 网页。
2. 点击扩展图标唤出悬浮面板，Token 会自动提取并填入。
3. 如果你在你想清理的频道内，可以直接点击「当前」按钮；或者点击「列表」，通过弹出的可视化界面选择服务器和频道。
4. 选择清理方向（从老到新 / 从新到老）和时间范围。
5. 勾选免责声明后点击「预览」。
6. 确认预览弹出的消息列表无误后，点击「删除预览」即可自动执行。

---

## 注意事项

- 消息删除后不可恢复，请务必先预览。
- 本扩展只能删除你账号自己发送的消息，无法删除他人的消息。
- 自动化请求违反 Discord TOS，仅限个人清理使用，请自行承担潜在的账号风险。
- 面板显示的 Token 代表你的账号完整权限，绝对不要截图发送给任何人。
