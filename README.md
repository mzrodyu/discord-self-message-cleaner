# Discord 自助冲水机 (Discord Self Message Cleaner)

**作者**: mzrodyu / catie

这是一个帮你快速、批量删除 Discord 中自己发过的历史消息的 Chrome 扩展。
告别手动一条条点击删除的痛苦，它会以一个优雅的悬浮面板注入到你的 Discord 网页里，随用随开，点完就删！

---

## 这是能干嘛的？

有时候我们在一些服务器或者私信里留下了太多历史消息，想删掉但又嫌麻烦。这个工具就是为了解决这个痛点的：

- **悬浮操作面板**：不用在扩展图标和页面之间切来切去，面板直接悬浮在 Discord 页面上，还能随便拖拽。
- **Token 全自动提取**：你不需要去找网络请求、复制 Token。只要你在 Discord 网页里，它就会自动把 Token 抓出来填好。
- **可视化服务器与频道选择**：不想手动抄 ID？点一下「列表」按钮，你的所有服务器和私聊列表都会弹出来，点选一下就自动填入。
- **花式冲水方向**：
  - **顺着冲（从老到新）**：非常适合清理几年前的历史遗留消息，不会把当前正在聊的最新消息给刷掉。
  - **倒着冲（从新到老）**：默认模式，从最近发的消息一路往回删。
- **时间范围精准把控**：支持设定起始和结束时间，点一下「同步最新」就能把当前时间卡死，配合预览功能，绝不会误删。
- **自动处理限速**：Discord 官方有 API 频率限制 (429 Too Many Requests)，它遇到了会自动停下来歇一会儿再继续删。

## 怎么安装？

支持 Chrome, Edge, Brave 等所有 Chromium 内核的浏览器。

1. 把这个项目下载或 clone 到你的电脑上。
2. 打开浏览器的扩展管理页面 (Chrome 浏览器输入 chrome://extensions/)。
3. 记得打开右上角的「开发者模式」。
4. 点击「加载已解压的扩展程序」，然后选择你刚刚下载的这个文件夹（里面必须有 manifest.json 文件）。
5. 在 Discord 网页版里点一下右上角插件栏的图标，面板就会弹出来了。

(如果你改了代码，记得在扩展管理页点一下「重新加载」刷新插件)

---

## 怎么使用？

1. 在浏览器里打开任意 Discord 网页。
2. 点击扩展图标唤出悬浮面板，Token 会自动填上。
3. 如果你在你想删消息的频道里，可以直接点「当前」按钮，它会把 URL 里的 ID 抓下来。
4. 如果你在别的页面，点「列表」，对着弹出来的界面选你要删的服务器和频道。
5. 选好你想「从老到新」还是「从新到老」。
6. 选好时间，勾选免责声明，点「预览」。
7. 看看弹出来的消息列表对不对，没问题就点「删除预览」，然后等待运行完毕。

## 避坑指南与注意事项

- **后果自负**：这类自动化工具在严格意义上违反了 Discord 的服务条款（TOS）。虽然只删自己的消息通常没人管，但风险始终存在（账号被警告或封禁），请自行承担使用风险。
- **删了就没了**：删除操作是直接走 API 的，删掉的消息不可恢复，一定要先预览！建议第一次用的时候把数量调小一点（比如 10 条）试下水。
- **Token 安全**：面板里的 Token 是直接跟你的账号绑定的，绝对不要截图发给别人，否则别人可以随意登录你的账号。
- **权限**：本扩展只能删除你自己发的消息（普通用户没有权限删别人的消息）。
